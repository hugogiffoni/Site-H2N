A

Algoritmo
Uma sequência finita e bem definida de instruções lógicas para resolver um problema ou realizar uma tarefa específica. É a base para a criação de programas de computador.
API (Application Programming Interface)
Um conjunto de definições e protocolos que permite a diferentes softwares se comunicarem e trocarem dados. Pense nela como uma "ponte" que conecta sistemas.
Autenticação
Processo de verificar a identidade de um usuário, sistema ou dispositivo. Geralmente envolve a apresentação de credenciais (como nome de usuário e senha).

B

Backup
Cópia de dados armazenada em um local diferente do original, para recuperação em caso de perda, corrupção ou exclusão acidental dos dados primários.
Blockchain
Tecnologia de registro distribuído que armazena transações em "blocos" encadeados e criptograficamente seguros, tornando-os imutáveis e transparentes.

C

Cloud Computing
Entrega de recursos de computação sob demanda — desde aplicativos até armazenamento de dados e poder de processamento — pela internet, com pagamento conforme o uso. Veja também SaaS e IaaS.
Criptografia
Processo de codificar informações para protegê-las de acessos não autorizados. Transforma dados em um formato ilegível, que só pode ser decifrado com uma chave.
Cybersecurity
Conjunto de tecnologias, processos e controles projetados para proteger sistemas, redes e dados de ataques digitais, danos ou acessos não autorizados.
