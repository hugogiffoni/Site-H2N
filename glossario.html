A

Algoritmo
Uma sequência finita e bem definida de instruções lógicas para resolver um problema ou realizar uma tarefa específica. É a base para a criação de programas de computador.
API (Application Programming Interface)
Um conjunto de definições e protocolos que permite a diferentes softwares se comunicarem e trocarem dados. Pense nela como uma "ponte" que conecta sistemas.
Autenticação
Processo de verificar a identidade de um usuário, sistema ou dispositivo. Geralmente envolve a apresentação de credenciais (como nome de usuário e senha).

B

Backup
Cópia de dados armazenada em um local diferente do original, para recuperação em caso de perda, corrupção ou exclusão acidental dos dados primários.
Blockchain
Tecnologia de registro distribuído que armazena transações em "blocos" encadeados e criptograficamente seguros, tornando-os imutáveis e transparentes.

C

Cloud Computing
Entrega de recursos de computação sob demanda — desde aplicativos até armazenamento de dados e poder de processamento — pela internet, com pagamento conforme o uso. Veja também SaaS e IaaS.
Criptografia
Processo de codificar informações para protegê-las de acessos não autorizados. Transforma dados em um formato ilegível, que só pode ser decifrado com uma chave.
Cybersecurity
Conjunto de tecnologias, processos e controles projetados para proteger sistemas, redes e dados de ataques digitais, danos ou acessos não autorizados.

D

DevOps
Integra desenvolvimento e operações para acelerar entregas de software com qualidade.
Docker
Plataforma de containers para empacotar aplicativos e suas dependências.

E

Edge Computing
Processamento de dados mais próximo da origem para reduzir latência.
Escalabilidade
Capacidade de um sistema lidar com aumento de carga.

F

Front-end
Parte visual com a qual o usuário interage em um sistema.
Framework
Conjunto de ferramentas que facilita o desenvolvimento de software.

G

Git
Sistema de controle de versões usado no desenvolvimento de software.
Governança de TI
Estrutura que alinha a TI aos objetivos da organização.

H

HTML
Linguagem de marcação usada na criação de páginas web.
HTTP / HTTPS
Protocolos de comunicação utilizados na internet.

I

IA (Inteligência Artificial)
Simulação de processos humanos por máquinas.
Infraestrutura de TI
Conjunto de recursos de hardware, software e rede.

L

Linux
Sistema operacional de código aberto amplamente usado.
Load Balancer
Distribui o tráfego entre servidores para otimizar performance.

M

Machine Learning
Subcampo da IA que permite aprendizado automático a partir de dados.
Métricas de TI
Indicadores que avaliam o desempenho de sistemas e processos.

N

NoSQL
Banco de dados não relacional usado para dados não estruturados.

O

Open Source
Software de código aberto, com livre distribuição e modificação.
Orquestração
Coordenação automatizada de processos e serviços em TI.

P

Proxy
Servidor intermediário entre usuário e internet.
Patch
Correção de software para resolver erros ou vulnerabilidades.

Q

Query
Instrução usada para buscar dados em bancos SQL.
QoS (Quality of Service)
Técnica para priorizar tipos de tráfego em redes.

R

RAM
Memória de acesso aleatório usada em tempo real pelos sistemas.
REST
Padrão para APIs web baseado em recursos e métodos HTTP.
RAID
Técnica de armazenamento com redundância e/ou desempenho.

S

SaaS
Software como serviço entregue via navegador, como Gmail ou Google Docs.
SQL
Linguagem usada para consulta e gerenciamento de dados relacionais.
Scrum
Metodologia ágil para gerenciamento de projetos em sprints.

T

TCP/IP
Conjunto de protocolos que conecta computadores na internet.
Token
Identificador usado para autenticação segura em sistemas.

U

UI
Interface com a qual o usuário interage em um sistema.
URL
Endereço utilizado para acessar recursos online.

V

VPN
Rede privada virtual usada para conexão segura pela internet.
Versionamento
Controle de versões de arquivos ou código-fonte.

W

Wi-Fi
Tecnologia que conecta dispositivos sem fio a redes locais.
WebSocket
Protocolo de comunicação em tempo real entre cliente e servidor.

X

XML
Formato de dados legível por humanos e máquinas para troca de informações.

Y

YAML
Formato de arquivo usado para configuração, simples e legível.

Z

Zettabyte
Unidade de medida equivalente a um bilhão de terabytes.
